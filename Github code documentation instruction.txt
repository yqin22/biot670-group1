Development Environment Setup
IDE
1. VS Code - https://code.visualstudio.com/
Git (Windows)
1. Download and install Git - https://git-scm.com/download/win
2. Download and install Source Tree - https://www.sourcetreeapp.com/

Development Workflow with Git
1. Clone repository from github to local
2. Develop/update code in the locally
3. After complete development locally, upload code back to github 
  a. Using the following commands 
  i. Stage
  ii. Commit
  iii. Push
  b. The above commands can be performed in SourceTree.
